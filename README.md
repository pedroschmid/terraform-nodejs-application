# NODEJS | ECS | RDS

This project is built with terraform to create an infrastructure that includes:

- Network configuration (VPC, Subnets, Internet Gateway, NAT Gateway, Route tables and associations.)
- IAM policies and roles 
- ALB
- ECS
- RDS
